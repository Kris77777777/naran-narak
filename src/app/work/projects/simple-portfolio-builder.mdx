## Overview
NinjaKH Express is an e-commerce and delivery platform that allows Cambodian customers to shop for products from international platforms like Lazada and Shopee, while also offering a parcel forwarding service from countries like Thailand and China. The website enables users to browse products, place orders, track shipments, and manage their accounts, making cross-border shopping more convenient and accessible with local delivery support.

## Key Features
- **Data Listing**: All of the products listing in the NinjaKH Express's website are the latest from Lazada & Shopee website with accurate stock and products matrix.
- **Cross-Border Shopping**: Browse and buy products from platforms like Lazada and Shopee.
- **Parcel Forwarding Service**: Ship items from Thailand, China, and Vietnam to Cambodia.
- **Product Categories**: Organized product listings with search and filter functionality.
- **Order Tracking**: Track the status of your packages in real-time.
- **Promotions & Wallet System**: Offers cashback and discount campaigns for top-ups.

## Technologies Used
- **Website**: Built with modern web technologie to ensure the website handling data loading and easy to customize with the beatiful UI.
- **Backend and API**: Powered by a custom backend system to handle order processing, tracking, forwarding logic, and API integration with external marketplaces
- **Responsive**: Fully responsive design to ensure smooth browsing and shopping experience across mobile, tablet, and desktop devices.
