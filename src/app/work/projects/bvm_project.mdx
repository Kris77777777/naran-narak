## Overview
BVM Petroleum App serves logistics and fuel distribution operations. It connects fuel stations and delivery teams, enabling users to:
  - Log in securely with their credentials.
  - Place and track fuel orders.
  - Monitor delivery status in real time.
  - Review history and generate reports.
The app is primarily designed for internal operations and business clients rather than public retail users.

## Key Features
- **Order Management**: Place fuel orders with details like quantity and recipient.
- **Delivery Status**: Live tracking of delivery stages (departed, arrived, offloaded, done).
- **Notifications**: Real-time updates for order status and changes.
- **Filtering Options**: Search or filter orders by various criteria.
- **Reporting Module**: Access categorized reports for better data insights.

## Technologies Used
- **Mobile Application**: Developed using advanced mobile app architecture for high performance and reliability
- **Backend API**: Capable of handling multiple concurrent users with smooth real-time operation
- **API**: Integrated with map services to track vehicle locations and calculate travel distances
- UX**/UI Design**: Designed with secure access and responsive UI to support logistics workflow efficiently
